# SAMSUNG SUPPLY CHAIN & LOGISTICS DASHBOARD

## Problem Statement
This dashboard analyzes Samsung’s supply chain operations, inventory management, shipment performance, supplier efficiency, and product sales trends. It helps businesses monitor operational performance, optimize inventory levels, and improve logistics decision-making using data-driven insights.

# Tech Stack & Development Process

## Technologies Used
- Power BI
- Power Query
- DAX
- SQL
- Data Modeling
- Interactive Visualizations
- AI-Assisted Dashboard Design

---

## Development Workflow
- Collected and structured supply chain, shipment, inventory, and product datasets.
- Performed data cleaning and transformation using **Power Query**.
- Built KPIs and calculated measures using **DAX**.
- Designed star schema relationships for optimized reporting.
- Created interactive dashboards with drill-through navigation and filters.
- Used AI tools for dashboard layout design, color optimization, and UI enhancement.
